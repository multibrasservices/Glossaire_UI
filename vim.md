Éditeur : Vim.
`:` : Mode commande.
`w` : write (sauvegarder).
`q` : quit (quitter).

####

C'est une commande de l'éditeur de texte Vim, l'éditeur par défaut de Git dans votre configuration.

Il faut être explicite avec ces outils.

*   **`:`** passe Vim en mode commande.
*   **`w`** est la commande pour *write* (écrire), c'est-à-dire sauvegarder le fichier.
*   **`q`** est la commande pour *quit* (quitter).

`:wq` est la combinaison directe et efficace pour dire "sauvegarde et quitte". C'est la seule chose à faire pour finaliser le commit.
